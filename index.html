Term 2 Week 1 - Making Websites! 12/1/18

Term 2 is all about HTML and CSS...

Your challenge today is to start making your first webpage. To do this we first need to learn about html and css. That means we need to learn all about TAGS.

To get started we are going to use Trinket - a great web based tool for learning how to write html (the writing on the page) and css (the styling of the page). We will start by doing the Birthday Card project.

If you want to stick with Scratch we still have another 6 great projects on the code club site including: Memory, Dodgeball, Brain Game, Catch the Dots, Clone Wars and Create your own world! - remember if you complete all 6 you get a certificate!

Don't forget to add your work to the OLM Code Club Studio: https://scratch.mit.edu/studios/3456235/
