Week 5 - Chatbot 19/10/17

Using IF ... THEN ... statements to make decisions in a program.

Your challenge today is to learn how to use:

  • TEXT INPUT - so you can make the program ask for information like your name or favourite animal
  • IF THEN statements - so you can make decisions in your program
IF THEN statements are also called conditional statements.

In CodeClub we will play a game where the whole class will become a computer program to show how we can divide the class into two groups (IF you were born before September 2007 THEN stand over here) and how we can use IF THEN ELSE to control the movement of a class of robots.

Jump to the Code Club Projects page and open up Chatbot.

Don't forget to add your work to the OLM Code Club Studio: https://scratch.mit.edu/studios/4318422/
