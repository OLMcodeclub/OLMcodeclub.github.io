Week 1 - Welcome to CodeClub! 15/09/17

What do we do at Code Club?

We will be making animations, games, websites, and much more. Code Club is all about using computers to help us make cool stuff. A key thing you will learn is how to program computers to make them follow your instructions.

What is the plan for the first term?

In term one we will be covering:

  • Scratch Module 1 - 6 great games to cover the basics of what you need to know to make your own game, animation, adventure story etc.
  • Scratch Module 2 - 6 more challenges which end in you making your own game
  • BBC Micro:bit - computers come in all shapes and sizes
  • A Christmas special - what will we make this year?

For the experienced Code Clubber you can join in with the above (note there are some new scratch challenges on the Code Club website) or you can:

  • Try the new Python Module - it is ace!
  • Build a website for Code Club or school (e.g. a weather web page, or a library database)
  • Create your own project using a makey makey, or make a robot with the BBC Micro:bit, or make models for 3D printing, or build a copy of OLM in Minecraft...

Let's get started!

A quick introduction to the 3 S's of Scratch: stage - sprites - scripts.

TIP FOR THE DAY: coding is all about giving computers instructions so that they do the task you want them to - we call these instructions ALGORITHMS (but you might also know them as recipes). The worksheets will really help you learn how to program.

2 tasks to get started:

  1. Log in to Scratch (we will help if you have not done this before)
  2. Make your club badge - real first name, scratch id, year group, robot picture of yourself
  3. Add your scratch name to the activity sheet on DrDunc's desk.
