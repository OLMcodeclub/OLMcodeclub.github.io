Week 15 - How to make Pancakes 24/2/17

Lets create some recipes to add to the OLM website.

Pancake Day, or Shrove Tuesday, is the traditional feast day before the start of Lent on Ash Wednesday. Next Tuesday parents at OLM will all be turning to Google and entering the phrase "how to make pancakes". Lets help them out by making some really interesting Pancake recipes!

Todays challenge:

Using Trinket - start to create your own recipe that we can add to the OLM web page. The Recipe project provides you with all the instructions you should need. The new tags you will need to learn are the:

  • <ul>
  • <ol>
  • <li>
list tags. You will also learn about using colours in CSS (names & hexadecimal codes). Enjoy!

If you want to stick with Scratch we still have another 6 great projects on the code club site including: Memory, Dodgeball, Brain Game, Catch the Dots, Clone Wars and Create your own world! - remember if you complete all 6 you get a certificate!

Don't forget to add your work to the OLM Code Club Studio: https://scratch.mit.edu/studios/3456235/
