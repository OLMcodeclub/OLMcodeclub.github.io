Week 21 - Recursive Challenge 19/5/17

How can we keep drawing the same thing?

In programming we have a really interesting concept called RECURSION which is when part of an an algorithm calls itself as part of its code. The BLOCKS function in Scratch is really useful for this. So for example I could write an algorithm to draw a box using PEN and MOVE and then at the end of the function I could ask it to draw another box using the same BLOCK. Games designers and artists use this type of algorithm to draw repeating objects like trees or scenery. You may have heard of the idea of FRACTALS which are natures forms of recursion.

Here is an example of a recursive box in scratch: https://scratch.mit.edu/projects/162013720/#editor
