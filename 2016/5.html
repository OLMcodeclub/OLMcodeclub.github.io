Week 5 - Chatbot 4/11/16

Using IF ... THEN ... statements to make decisions in a program.

Your challenge today is to learn how to use:

  • TEXT INPUT - so you can make the program ask for information like your name or favourite animal
  • IF THEN statements - so you can make decisions in your program
IF THEN statements are also called conditional statements. In CodeClub we will play a game where the whole class will become a computer program to show how we can divide the class into two groups (IF you were born before September 2006 THEN stand over here) and how we can use IF THEN ELSE to control the movement of a class of robots.
