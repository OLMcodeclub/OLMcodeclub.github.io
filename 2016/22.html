Week 22 - Physical Computing 06/06/17

Computing with buttons, lights, microphones, buzzers etc.

Physical computing means building interactive physical systems using software and hardware that can sense and respond to the real world. So far in scratch we have been building programmes that we interact with through the computer screen. Today we are going to start using the fantastic BBC Micro:bit

As a first step open this webpage: https://pxt.microbit.org/ and start to explore the different blocks you can use. See what you can build in 10 minutes.

Todays challenge:

Can you make a programme to keep score each time a button is pressed? How can you let someone know what the current score is? You will get bonus points if you can work out how to reset the score without restarting the device.
